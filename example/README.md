# pcanvas_flutter example

This project is a `pcanvas_flutter` usage example.

## Running the Example

To run the example run the following commands:
```shell
cd pcanvas_flutter/example
flutter pub get
flutter run
```

Then select your preferred device to run the Flutter app.
